# analise-vendas-python

Projeto de Análise de Dados desenvolvido em Python com o objetivo de demonstrar habilidades iniciais em Data Analytics

---

I. Objetivo do Projeto
Realizar uma análise simples de dados de vendas, simulando um cenário real de negócio, para extrair métricas importantes e gerar visualizações que auxiliem na tomada de decisão.

---

II. O que foi analisado
- Faturamento total
- Produto mais vendido (em quantidade)
- Faturamento por mês
- Visualização gráfica dos resultados

---

III. Tecnologias Utilizadas
- **Python**
- **Pandas** – manipulação e análise de dados
- **Matplotlib** – visualização de dados
- **Jupyter Notebook**

---

IV. Estrutura do Projeto
analise-vendas-python/
├── 01_analise_vendas.ipynb
└── README.md

---

V. Conjunto de Dados
O dataset utilizado é **fictício**, criado apenas para fins educacionais, contendo:
- Data da venda  
- Produto  
- Quantidade vendida  
- Preço unitário
